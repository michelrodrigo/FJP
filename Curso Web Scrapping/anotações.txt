web scrapping (raspagem) é a coleta de dados disponíveis em um site e sava-los em um formato que é mais fácil de usarmos

pode ser usado para o bem e para o mal

já fazemos no dia a dia, cc + cv, porém é manual, queremos fazer de forma automática.

web scrapping normalmente é dividio em duas partes, uma que busca na internet os sites que tem a informação que procuramos e a segunda é que retira os dados dos sites

webscrapping é muito dependente do site, não tem receita

alguns sites tem mecanismos para evitar, como captcha. não iremos trabalhar. Outro problema é o limite de número de acessos de um site.


webscrapping tem duas abordagens:

1) usada quando os dados estão no front end, estão vísiveis

2) quando o site tem uma api que faz requisições a um servidor.
